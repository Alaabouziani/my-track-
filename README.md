# MonTrack POS

A modern, mobile-responsive Point of Sale (POS) and Inventory Management system built with React and Vite.

## Key Features

- **Sales Interface:** Streamlined checkout process with a dynamic cart.
- **Inventory Management:** Track products, stock levels, and pricing in real-time.
- **Client & Store Management:** Manage customer details and distribution points.
- **Transaction History:** View past sales and generate receipts.
- **Dark Mode Support:** Built-in theme switching for better usability.
- **Customizable Settings:** Dynamic configuration for application and distribution names.

## Getting Started

1.  Clone the repository:
    ```bash
    git clone https://github.com/AlaaeddineBz/MonTrack-POS-.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Technologies

- React
- Vite
- Lucide React (Icons)
